## Calculation Examples

### Example 1: Simple Item
```
Inputs:
  Qty = 100
  Purchase Price = вӮ№5.00
  Discount% = 0
  GST% = 12

Calculations:
  Line Subtotal = 100 Г— 5.00 = вӮ№500
  Line Discount = 500 Г— 0% = вӮ№0
  Taxable Amount = 500 - 0 = вӮ№500
  GST Amount = 500 Г— 12% = вӮ№60
  Line Total = 500 + 60 = вӮ№560
```

### Example 2: With Discount
```
Inputs:
  Qty = 50
  Purchase Price = вӮ№10.00
  Discount% = 10
  GST% = 12

Calculations:
  Line Subtotal = 50 Г— 10.00 = вӮ№500
  Line Discount = 500 Г— 10% = вӮ№50
  Taxable Amount = 500 - 50 = вӮ№450
  GST Amount = 450 Г— 12% = вӮ№54
  Line Total = 450 + 54 = вӮ№504
```

### Example 3: Multiple Items with Header Discount
```
Item 1:
  Line Subtotal = вӮ№500, GST = вӮ№60, Line Total = вӮ№560

Item 2:
  Line Subtotal = вӮ№300, GST = вӮ№36, Line Total = вӮ№336

Item 3:
  Line Subtotal = вӮ№200, GST = вӮ№24, Line Total = вӮ№224

Summary:
  Total Subtotal = 500 + 300 + 200 = вӮ№1,000
  Total GST = 60 + 36 + 24 = вӮ№120
  Subtotal + GST = 1,000 + 120 = вӮ№1,120
  Header Discount = вӮ№50
  Net Amount = 1,120 - 50 = вӮ№1,070
  
  If Paid = вӮ№1,000
  Due Amount = 1,070 - 1,000 = вӮ№70
```

## User Interaction Flow

```
1. Page Loads
   в””в”Җ Suppliers & medicines fetched from API

2. User Selects Supplier
   в””в”Җ Details auto-populate

3. User Adds First Item
   в””в”Җ New row appears

4. User Selects Medicine
   в””в”Җ Details auto-fill (generic, company, GST%)

5. User Enters Batch & Dates
   в””в”Җ Row data updates

6. User Enters Qty & Rate
   в””в”Җ Amount calculates instantly
   в””в”Җ Summary updates

7. User Adjusts Discounts/Tax
   в””в”Җ Amount recalculates
   в””в”Җ Summary updates

8. User Clicks Row
   в””в”Җ Details panel shows
   в””в”Җ Expandable row shows

9. User Adds More Items
   в””в”Җ Repeat 3-8

10. User Reviews Summary
    в””в”Җ Checks calculations
    в””в”Җ Enters paid amount
    в””в”Җ Sees due amount

11. User Clicks Save
    в””в”Җ Validation runs
    в””в”Җ If valid вҶ’ API call
    в””в”Җ Success message
    в””в”Җ Form resets
```
